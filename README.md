# Set up vim
```
# Download files
git clone https://github.com/braceal/vim-config.git

# Install vundle
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

# Copy colorscheme and vimrc
mkdir -p ~/.vim/colors
cp vim-config/monokai.vim ~/.vim/colors/
cp vim-config/.vimrc ~/.vimrc
```
Open vim and enjoy.
